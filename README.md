# 🏀 NBA Performance Analytics Dashboard

An interactive Power BI dashboard analyzing team and player performance in the NBA. The project highlights scoring trends, win-loss patterns, and player efficiency through intuitive and dynamic visualizations.

---

## 📊 Project Overview
This dashboard enables users to explore NBA data in a structured and visually engaging way, helping uncover meaningful insights across teams and players.

---

## 🚀 Features
- Performance trends over time  
- Team comparison (wins, losses, key stats)  
- Player-level insights and efficiency metrics  
- Interactive filters (team, player, season)  
- Clean and intuitive visualizations  

---

## 🛠️ Tools & Technologies
- Power BI  
- Data Cleaning & Transformation  
- Data Analysis & Visualization
